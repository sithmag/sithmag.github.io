# Sithmag.github.io
## developing

create a devcontainer - `CMD + Shift + P` and select `Dev Containers: Reopen in Container`, or rebuild the container if it is already opened.

then run:

```bash
# jekyll via ruby and bundler
bundle install
bundle exec jekyll serve --livereload
```
